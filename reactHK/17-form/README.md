## 🧑‍💻 How to Run Locally

### 1. Clone the repo

```bash
git clone https://github.com/asyncPranav/Learning-WebDev/tree/main/reactHK/17-Form.git
```

### 2. Move to directory

```bash
cd 17-Form
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```
